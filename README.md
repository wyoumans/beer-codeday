# Beer CodeDay

Built on Laravel 6 & Vue 2

## Getting Started

### DB Setup

Run the migrations:
```
php artisan migrate
```

Seed the database:
```
php artisan db:seed
```

### Assets

Install dependencies
```
composer install
npm install
```

Compile Assets
```
npm run dev
```

Watch Assets
```
npm run watch
```

Compile assets for deployment
```
npm run production
```

### Tests

Run all unit and feature tests locally
```
sh tests.sh
```
